1.2 ODE Revision¶

Separable ODEs¶

We say that an ODE is separable if it can be written in the form

(1)¶\[ \frac{\mathrm{d}x}{\mathrm{d}t} = \frac{f(t)}{g(x)} \]

To solve such an equation we rearrange and integrate as follows

\[ \int g(x)\mathrm{d}x = \int f(t)\mathrm{d}t. \]

Example 1.2.1 Solve \(\displaystyle \frac{\mathrm{d}x}{\mathrm{d}t} = -2xt\).

The right-hand side is clearly separable with \(\displaystyle f(t)=-2t\) and \(\displaystyle g(x)=1/x\) and so

\[ \int x^{-1}\mathrm{d}x = \int (-2t)\mathrm{d}t \]

thus

\[ \log{x} = -t^2 + c \]

giving

\[ x(t) = e^{-t^2+c} = ke^{-t^2}. \]

Example 1.2.2 Solve \(\displaystyle \frac{\mathrm{d}x}{\mathrm{d}t} = \frac{t}{x}\).

Multiplying through by \(x\mathrm{d}t\) and integrating gives

\[ \int x\mathrm{d}x = \int t\mathrm{d}t. \]

Then

\[ \frac{1}{2}x^2 = \frac{1}{2}t^2 + \frac{c}{2} \]

and so

\[ x^2 = t^2 + c. \]

Taking square roots we obtain the general solution

\[ x = \pm\sqrt{t^2 + c}. \]

Note

If the differential equation is of the form

(2)¶\[ \frac{\mathrm{d}x}{\mathrm{d}t} =f(x/t) \]

then we can use a change of variables \(x=vt\) to rewrite (2) as

\[ v+t\frac{\mathrm{d}v}{\mathrm{d}t} = f(v), \]

which is a separable equation (i.e. we can solve as in our earlier examples)


Example 1.2.3 Solve \(\displaystyle \frac{\mathrm{d}x}{\mathrm{d}t} = \frac{t+x}{t}\).

The right-hand side can be written as \(\displaystyle 1 + x/t\) and hence is of the form \(\displaystyle f(x/t)\).

Putting \(x = vt\) we get

\[ v+t\frac{\mathrm{d}v}{\mathrm{d}t} = 1+v \]

or

\[ t\frac{\mathrm{d}v}{\mathrm{d}t} = 1. \]

Multiplying through by \(\displaystyle \mathrm{d}t/t\) and integrating gives

\[ \int \mathrm{d}v = \int t^{-1}\mathrm{d}t \]

or

\[ v = \log{t} + \log{k} = \log{kt}. \]

Finally, putting \(v=x/t\), we obtain

\[ x(t) = t\log{kt}. \]

Integrating Factor¶

Given a first-order linear differential equation of the form

(3)¶\[ x' + p(t)x = q(t) \]

where \(p\) and \(q\) are functions of \(t\) only, we can use the so-called integrating factor method to solve.

To do this we introduce an integrating factor as follows:

\[ f(t) = e^{\int p(t)\mathrm{d}t}. \]

We then multiply (3) by the function \(f(t)\) to obtain

\[ x'e^{\int p(t)\mathrm{d}t} + p(t)e^{\int p(t)\mathrm{d}t}x = q(t)e^{\int p(t)\mathrm{d}t}. \]

Note

This trick is just the product rule in reverse!

The trick to solving is to notice that the left-hand side of the above equation can be rewritten as

\[ x'e^{\int p(t)\mathrm{d}t} + p(t)e^{\int p(t)\mathrm{d}t}x = \frac{\mathrm{d}}{\mathrm{d}t}\left(x e^{\int p(t)\mathrm{d}t}\right). \]

Using this (3) becomes

\[ \frac{\mathrm{d}}{\mathrm{d}t}\left(x e^{\int p(t)\mathrm{d}t}\right) = q(t)e^{\int p(t)\mathrm{d}t}, \]

which can be integrated and solved to determine \(x(t)\).

Hopefully an example will clarify how the above process works in practice.


Example 1.2.4 Solve \(\displaystyle x' = t + \frac{2x}{t}\).

To start let us rearrange the equation into the standard form given in (3)

\[ x' - \frac{2}{t}x = t \]

so that \(p(t)=-2/t\) and \(q(t)=t\).

It follows that the integrating factor

\[ f(t) = e^{\int (-2/t)\mathrm{d}t} = e^{-2\log|t|} = e^{\log(t^{-2})} = t^{-2}. \]

Multiplying by the integrating factor gives

\[ t^{-2}x' - \frac{1}{t^3}x = t^{-1}. \]

Thus

\[ \frac{\mathrm{d}}{\mathrm{d}t}\left(xt^{-2}\right) = t^{-1}, \]

which integrates to

\[ xt^{-2} = \int t^{-1}\mathrm{d}t = \log|t| + \log|k| = \log|kt| \]

giving

\[ x(t) = t^2\log|kt|. \]

Complementary Functions for Second-Order ODEs¶

We wish to solve equations of the form

(4)¶\[ x'' -px' + qx = 0, \]

with \(p, q\in\mathbb{R}\).

Experience suggests that solutions of the above are likely to be of the form \(\displaystyle x(t) = e^{\lambda t}\) and so we substitute this into (4) as a trial solution obtaining

(5)¶\[\begin{split} \lambda^2e^{\lambda t} -p\lambda e^{\lambda t} + q\lambda^2e^{\lambda t} &= e^{\lambda t}\left(\lambda^2-p\lambda+q\right) = 0\\ \implies \left(\lambda^2-p\lambda+q\right) = 0 \end{split}\]

Equation (5) is called the auxillary equation of the ODE in (4). It is a quadratic equation for \(\lambda\) and hence has two roots, \(\lambda_1\) and \(\lambda_2\), say, so that if these are distinct, there are two possible solutions of (4),

\[ x_1(t) = e^{\lambda_1 t}\quad\text{and}\quad x_2(t) = e^{\lambda_2 t}. \]

Note

Since the equation is linear we can combine solutions (linearly!) to determine the general solution

Thus a general solution of (4) is

\[ x(t) = Ae^{\lambda_1 t} + Be^{\lambda_2 t} \]

provided \(\lambda_1\neq\lambda_2\).


Example 1.2.5 (distinct roots) Solve \(\displaystyle x''-3x'+2x=0\).

Substituting \(\displaystyle x=e^{\lambda t}\) into our ODE gives the following auxillary equation

\[ \lambda^2-3\lambda+2=0, \]

which has roots 1 and 2. The complementary function is therefore

\[ x(t) = Ae^t + Be^{2t}. \]

Example 1.2.6 (repeated roots) Solve \(\displaystyle x''-4x'+4x = 0.\)

The axillary equation is

\[ \lambda^2-4\lambda+4= (\lambda -2)^2 = 0. \]

Thus we have obtained just one solution: \(\displaystyle e^{2t}\).

A second solution is provided by \(\displaystyle x=te^{2t}\) (the reason for this will be explained later on in the course). To check this is indeed a solution note that \(\displaystyle x'=(1+2t)e^{2t}\) and \(\displaystyle x''=4(1+t)e^{2t}\) so that

\[ (4(1+t)-4(1+2t)+4t)e^2t = 0. \]

Thus \(\displaystyle x=te^{2t}\) solves our equation.

Forming a linear combination of our two solutions provides the following general solution

\[ x(t) = (A+Bt)e^{2t}. \]

Example 1.2.7 (complex roots) Solve \(\displaystyle x''-6x'+25x=0\).

The auxillary equation is given by

\[ \lambda^2-6\lambda+25=0 \]

whose roots are \(\lambda_{1,2} = 3\pm 4i\). These are distinct so we can write

\[ x(t) = Ae^{(3+4i)t} + Be^{(3-4i)t}. \]

It appears that our solution is complex in this case; however, let us put

\[ e^{(3\pm 4i)t} = e^{3t}e^{\pm 4it} = e^{3t}\left(\cos{4t}\pm i\sin{4t}\right) \]

so that the solution becomes

\[\begin{split} x(t) &= e^{3t}(A(\cos{4t}+i\sin{4t})+B(\cos{4t}-i\sin{4t}))\\ &= e^{3t}(C\cos{4t}+D\sin{4t}), \end{split}\]

where \(\displaystyle C=A+B\) and \(D=(A-B)i\) are arbitrary constants.

Clearly for real initial conditions the solutions are real for all \(t\).


Recall that we can use the same technique to solve an inhomogeneous ODE of the form

(6)¶\[ x''-px'+qx = f(t); \]

however, we must now find a particular solution, \(x_p(t)\), to accompany our complementary solution, \(x_c(t)\). This leads to a general solution of (6) of the form

\[ x(t) = x_c(t) + x_p(t). \]

Example 1.2.8 (inhomogeneous) Solve \(\displaystyle x''+3x'-4x=6t\).

We start by solving the homogeneous equation

\[ x_c''+3x_c-4x_c=0. \]

The auxillary equation is \(\displaystyle \lambda^2+3\lambda-4=0\) which has roots 1 and -4 so that

\[ x_c(t) = Ae^t + Be^{-4t}. \]

Since the right-hand side of our ODE is linear in \(t\) we try the particular solution \(x_p = at+b\), which gives \(x_p'=a\) and \(x_p''=0\). Putting these into our ODE, we find

\[ 0+3a-4(at+b) = 6t. \]

Since this equation must hold for all \(t\), we have that

\[ 3a-4b = 0 \quad \text{and} \quad -4a = 6, \]

or \(a=-\frac{3}{2}\) and \(b=-\frac{9}{8}\) giving the following particular integral

\[ x_p(t) = -\frac{3}{2}t-\frac{9}{8} \]

Finally we have that the general solution is given by

\[\begin{split} x(t) &= x_c(t) + x_p(t)\\ &= Ae^t + Be^{-4t}-\frac{3}{2}t-\frac{9}{8}. \end{split}\]

As usual to find the arbitrary constants \(A\) and \(B\) requires additional information (i.e. initial or boundary data).


Warning

In general, when choosing the form of the particular solution, \(x_p(t)\), we are guided by the function \(f(t)\) on the right-hand side of (6); however, care must be taken if the right-hand side function, \(f(t)\), is similar to the complementary function \(x_c(t)\).
