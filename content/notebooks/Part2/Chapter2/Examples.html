2.3 Examples¶

Analytic functions¶

Example 2.3.1 Show that \(f(z)=z^2\) is analytic everywhere.

Solution

\[ f(z)=z^2=(x+iy)^2=x^2-y^2 + 2ixy \]

By inspection, \(u(x,y)=\mathrm{Re}(f)=x^2-y^2\) and \(v(x,y)=\mathrm{Im}(f)=2xy\) and the first derivatives are given by:

\[ \frac{\partial u}{\partial x}= 2x,\ \frac{\partial v}{\partial x}= 2y,\ \frac{\partial u}{\partial y}= -2y,\ \frac{\partial v}{\partial y}= 2x \]

Clearly the Cauchy-Riemann equations are satisfied and so the function is analytic. Recalling what we did above, the derivative is given by

\[ \frac{\mathrm{d} f}{\mathrm{d} z}=\frac{\partial u}{\partial x}+i\frac{\partial v}{\partial x}=-\frac{\partial v}{\partial y}+i\frac{\partial u}{\partial y}=2x+2iy=2z. \]

Example 2.3.2 Show that the function \(f(z)=\overline{z}\) is not analytic anywhere.

Solution

\[ f(z)=\overline{z}\ \mbox{so}\ u=x,\ v=-y \]

Differentiating, we obtain

\[ \frac{\partial u}{\partial x}= 1,\ \frac{\partial v}{\partial x}= 0,\ \frac{\partial u}{\partial y}= 0,\ \frac{\partial v}{\partial y}= -1 \]

So the Cauchy-Riemann equations are not satisfied.


Example 2.3.3 Show that Any function of the form \(f=f(z,\overline{z})\) is not analytic

Solution This can be shown with little difficulty by treating \(z\) and \(\overline{z}\) as independent variables and using the chain rule of differentiation:

\[ \frac{\mathrm{d}}{\mathrm{d} z}f(z,\overline{z}) = \frac{\partial f}{\partial z}\frac{\mathrm{d} z}{\mathrm{d} z}+\frac{\partial f}{\partial \overline{z}}\frac{\mathrm{d} \overline{z}}{\mathrm{d} z} \]

Now, \(\frac{\mathrm{d} \overline{z}}{\mathrm{d} z}\) does not exist (see above!). The function \(f(z,\overline{z})\) is therefore not analytic.


Harmonic functions¶

Example 2.3.4 Show that the the function \(f(z)=z^3\) is harmonic

Solution

Writing in terms of \(x\) and \(y\):

\[ f(z)=(x+iy)^3=(x^3-3xy^2) + i(3x^2y-y^3) = u(x,y)+iv(x,y). \]

Then, differentiating:

\[\begin{split} \frac{\partial u}{\partial x}&=&3x^2-3y^2 \Rightarrow \frac{\partial^2 u}{\partial x^2}=6x,\\ \frac{\partial u}{\partial y}&=&-6xy \Rightarrow \frac{\partial^2 u}{\partial y^2}=-6x,\\ \frac{\partial v}{\partial x}&=&6xy \Rightarrow \frac{\partial^2 v}{\partial x^2}=6y,\\ \frac{\partial v}{\partial y}&=&3x^2-3y^2 \Rightarrow \frac{\partial^2 v}{\partial y^2}=-6y, \end{split}\]

So \(\nabla^2u=\nabla^2v=0\) and the function is harmonic.


Finding a function¶

Example 2.3.5 If \(u(x,y)=x(1-y)\), find \(v\)

Solution

Firstly, \(\frac{\partial^2 u}{\partial x^2}=\frac{\partial }{\partial x}(1)=0\) and \(\frac{\partial^2 u}{\partial y^2}=\frac{\partial }{\partial y}(-x)=0\) so \(u\) satisfies Laplace’s equation and may therefore be taken as the real part of an analytic function.

Then Cauchy-Riemann:

\[ \frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}=1-y\ \mbox{and}\ \frac{\partial v}{\partial x} = -\frac{\partial u}{\partial y}=x. \]

Integrating each of the above to determine \(v\) provides:

\[ v=\int 1-y \mathrm{d} y = y -\frac{y^2}{2} + h(x)\ \mbox{and}\ v=\int x \mathrm{d} x=\frac{x^2}{2} + g(y). \]

Equating the two expressions gives: \(h(x)=\frac{x^2}{2}\) and \(g(y)=y-\frac{y^2}{2}\) so the function is:

\[ f(z) = x(1-y) + i(\frac{x^2-y^2}{2}+y+k). \]

Note that this is still only determined up to an additional arbitrary constant \(k\) (Exercise: do the differentiation of \(v\) to show why)


Expressing a function in terms of \(z\) given \(x\) and \(y\)¶

Example 2.3.6 \(f_1=2x+3+2iy+4i\), \(f_2=e^{-2x}\left(\cos(2y)-i\sin(2y)\right)\), \(f_3=x-iy\); express \(f_1,\ f_2\) and \(f_3\) in terms of \(z\)

Solution

In each case, taking \(y=0\) gives

\[ f_1(x)=2x+3+4i,\quad f_2(x)=e^{-2x},\quad f_3(x)=x \]

and so we obtain:

\[ f_1(z)=2z+3+4i,\quad f_2(z)=e^{-2z},\quad f_3(z)=z \]
