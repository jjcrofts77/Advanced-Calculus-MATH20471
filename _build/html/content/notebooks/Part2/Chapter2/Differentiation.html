2.1 Differentiation

2.1 Differentiation#

The definition of the complex derivative is similar to that of the derivative of a real-valued function.

Definition 11 (complex derivative )

If \(f(z)\) (\(z=x+iy\)) is a complex function, then \(f\) is said to be differentiable at a point \(z\) if:

\[ \lim_{\Delta z\to 0} \frac{f(z+\Delta z)-f(z)}{\Delta z} \]

exists and is independent of the manner that \(\Delta z\) approaches zero.

There are of course an infinite number of ways (some of which are shown in the applet below) with which we can approach \(z_0\) in the above limit. As noted above, the result should not depend on the choice of approach.

Denoting the point of interest by \(z_0\) (and defining \(z-z_0=\Delta z\) in the normal way), an alternative and possibly more familiar notation is:

\[ \frac{\mathrm{d}f}{\mathrm{d} z}\bigg|_{z=z_0}=\lim_{z\to z_0}\frac{f(z)-f(z_0)}{z-z_0}. \]

Given that \(z\) lies in the complex plane, there are many different ways which \(z\) can approach \(z_0\). Again, \(\tfrac{\mathrm{d} f}{\mathrm{d} z}\) must be independent of these choices.


We consider two separate cases:

  1. \(z\rightarrow z_0\) along the line \(z=x+iy_0\):

    We approach \(z_0\) along the line \(y=y_0\); i.e. \(z\rightarrow z_0\) implies \(x\rightarrow x_0\). Then, using the above definition of the derivative

\[\begin{split} \frac{\mathrm{d} f}{\mathrm{d} z}\bigg|_{z=z_0}&=\lim_{z\to z_0}\frac{f(z)-f(z_0)}{z-z_0}\\ &=\lim_{z\to z_0}\frac{u(x,y_0)+iv(x,y_0) - u(x_0,y_0)-iv(x_0,y_0)}{x-x_0}\\ &= \lim_{z\to z_0}\frac{u(x,y_0)-u(x_0,y_0)}{x-x_0} +i\frac{ v(x,y_0)-v(x_0,y_0)}{x-x_0}\\ &= \frac{\partial u}{\partial x}+i\frac{\partial v}{\partial x} \end{split}\]

  1. \(z\rightarrow z_0\) along the line \(z=x_0+iy\):

    We approach \(z_0\) along the line \(x=x_0\); i.e. \(z\rightarrow z_0\) implies \(y\rightarrow y_0\). Then, using the above definition of the derivative

    \[\begin{split} \frac{\mathrm{d} f}{\mathrm{d} z}\bigg|_{z=z_0}&=\lim_{z\to z_0}\frac{f(z)-f(z_0)}{z-z_0}\\ &=\lim_{z\to z_0}\frac{u(x_0,y)+iv(x_0,y) - u(x_0,y_0)-iv(x_0,y_0)}{i(y-y_0)}\\ &= \lim_{z\to z_0}-i\frac{u(x_0,y)-u(x_0,y_0)}{y-y_0} +\frac{ v(x_0,y)-v(x_0,y_0)}{y-y_0}\\ &= \frac{\partial v}{\partial y}-i\frac{\partial u}{\partial y}\end{split}\]

The same method works employing the \(\Delta z\rightarrow 0\) notation (Recommend you do this as an exercise).

Now, comparing real and imaginary parts, we find that in order for the expressions to be equal, the Cauchy-Riemann conditions must be satisfied:

(51)#\[ \frac{\partial u}{\partial x}=\frac{\partial v}{\partial y}\quad \frac{\partial v}{\partial x}=-\frac{\partial u}{\partial y} \]

These equations are key to complex analysis and have a number of important ramifications. Above, we have shown that they are necessary conditions for functions of a complex variable to be analytic (i.e. differentiable). Note that the converse result is false, unless the partial derivatives in (51) are continuous.

Theorem 7 (Cauchy-Riemann equations)

A complex-valued fuinction

\[ f(z) = u(x,y)+iv(x,y) \]

is differentiable if and only if its real and imaginary parts are continuously differentiable and satisfy the Cauchy-Riemann equations given by

\[ u_x=v_y, \quad u_y=-v_x \]

In this case the complex derivative of \(f(z)\) is given by the following equivalent expressions

\[ f'(z) = u_x+iv_x = v_y-iu_y. \]

Fortunately, complex differentiation follows the same set of rules that we learnt for real-valued functions, and so, for example, we have the results

\[ \frac{\mathrm{d}}{\mathrm{d}z}z^n = nz^{n-1}, \quad \frac{\mathrm{d}}{\mathrm{d}z} e^{\alpha z} = \alpha e^{\alpha z}, \quad \frac{\mathrm{d}}{\mathrm{d}z}\log(z) = \frac{1}{z}, \quad etc. \]

It should also be clear that one can determine the derivatives of complex trigonometric and hyperbolic functions using Euler’s formula (exercise) as follows:

\[\begin{split} \frac{\mathrm{d}}{\mathrm{d}z} \sin{z} &= \cos{z}\quad \frac{\mathrm{d}}{\mathrm{d}z} \cos{z} &= -\sin{z}\\ \frac{\mathrm{d}}{\mathrm{d}z} \sinh{z} &= \cosh{z}\quad \frac{\mathrm{d}} {\mathrm{d}z} \cosh{z} &=-\sinh{z}. \end{split}\]

In addition to this, the formulae for computing the derivatives of sums, products, ratios and compositions remain the same! (i.e. no need to learn any new rules!)

Example 35

Consider the function \(f(z) = z^2-z\), which can be written

\[ z^2 = (x^2-x-y^2) + iy(2x-1). \]

Its real part \(u=x^2-x-y^2\) and imaginary part \(v=y(2x-1)\) satisfy the Cauchy-Riemann equations, since

\[ u_x = 2x-1 =v_y \quad \text{and} \quad v_x = 2y = -u_y. \]

Theorem 7 implies that \(f(z) = z^2-z\) is differentiable with derivative given by

\[ f'(z) = u_x+iv_x = v_y-iu_y = (2x-1) + 2iy = 2z-1. \]
