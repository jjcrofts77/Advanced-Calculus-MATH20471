2.2 Fourier Series for Functions of Period \(2\pi\)¶

Let \(f\) be a function of period \(2\pi\). We would like to get an expansion for \(f\) of the form

(2)¶\[ f(x) = \frac{1}{2}a_0 +\sum_{n=1}^\infty \left[a_n\cos(nx) + b_n\sin(nx)\right], \]

where the \(a_n\) and \(b_n\) are constants. Remember that \(\sin(nx)\) and \(\cos(nx)\) are periodic with period \(2\pi\).

Such an expansion is know as a Fourier series (FS).

We have two questions to answer about the above series:

Question 1: If the series on the RHS of the above equation converges, can we find constants \(a_0, a_n\) and \(b_n\) in terms of the function \(f\)?

Question 2: with these \(a_n, b_n\), for what values of \(x\in\mathbb{R}\) does our FS equation hold?

Question 1¶

Suppose the above equation is true and that we can integrate it term by term (\emph{i.e} the series on the RHS above converges uniformly) so that

\[ \int_{-\pi}^\pi f(x)\mathrm{d}x = \frac{1}{2}a_0\int_{-\pi}^\pi\mathrm{d}x +\sum_{n=1}^\infty \left[a_n\int_{-\pi}^\pi\cos(nx)\mathrm{d}x + b_n\int_{-\pi}^\pi\sin(nx)\mathrm{d}x\right], \]

since

\[ \int_{-\pi}^\pi \mathrm{d}x = 2\pi, \quad \int_{-\pi}^\pi \cos(nx)\mathrm{d}x = 0, \quad \int_{-\pi}^\pi\sin(nx)\mathrm{d}x=0, \]

we must have

\[ a_0 = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\mathrm{d}x. \]

Thus if equation (2) holds, then we know \(a_0\).


Lemma 2.2.1: Let \(n, m\in\mathbb{N}\backslash 0\). We then have the following equalities:

(3)¶\[ \int_{-\pi}^\pi \sin(mx)\cos(nx)\mathrm{d}x = 0, \]
(4)¶\[ \int_{-\pi}^\pi\sin(mx)\sin(nx)\mathrm{d}x = \pi\delta_{nm}, \]
(5)¶\[ \int_{-\pi}^\pi\cos(mx)\cos(nx)\mathrm{d}x = \pi\delta_{nm}. \]

where \(\delta_{mn}\) is the Kronecker delta defined by

\[\begin{split} \delta_{mn} = \begin{cases}0& n\neq m\\1&n=m.\end{cases} \end{split}\]

Proof

Equation (3) is trivial as \(\sin(mx)\cos(nx)\) is odd. For equation (4) we compute, for \(n\neq m\),

\[\begin{split} \int_{-\pi}^\pi\sin(mx)\sin(nx)\mathrm{d}x &= \frac{1}{2}\int_{-\pi}^\pi\left[-\cos((m+n)x)+\cos((m-n)x)\right]\mathrm{d}x\\ &= \frac{1}{2}\left[\frac{-\sin((m+n)x)}{m+n}+\frac{\sin((m-n)x)}{m-n}\right]_{-\pi}^\pi\\ &=0. \end{split}\]

If \(n=m\) we have

\[ \int_{-\pi}^\pi\sin(nx)\sin(mx)\mathrm{d}x = \int_{-\pi}^\pi\frac{1-\cos(2nx)}{2}\mathrm{d}x = \frac{1}{2}\left[x-\frac{\sin(2nx)}{2n}\right] = \pi \]

Similar computations yield equation (5). \(\quad\blacksquare\)

Thus to find \(a_n\) and \(b_n\), we assume equation (2) is true. Multiplying both sides by \(\cos(mx)\) and integrating term-wise gives

\[\begin{split} \int_{-\pi}^\pi f(x)\cos(mx)\mathrm{d}x =\frac{1}{2}a_0&\int_{-\pi}^\pi \cos(mx)\mathrm{d}x\\&+\sum_{n=1}^\infty a_n\int_{-\pi}^\pi\cos(nx)\cos(mx)\mathrm{d}x\\ &+\sum_{n=1}^\infty b_n\int_{-\pi}^\pi \sin(nx)\cos(mx)\mathrm{d}x. \end{split}\]

The first term on the RHS is equal to zero (even function) for \(m\neq 0\). Using Lemma 2.1 for the remaining terms gives

\[ \int_{-\pi}^\pi f(x)\cos(mx)\mathrm{d}x = \sum_{n=1}^\infty a_n\pi\delta_{nm} = \pi a_m, \]

and hence

\[ a_m = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\cos(mx)\mathrm{d}x. \]

Note that this also holds for \(m=0\) (which explains the factor of 1/2).

Multiplying equation (2) by \(\sin(mx)\) and integrating term-wise, we similarly obtain

\[ b_m = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\sin(mx)\mathrm{d}x. \]

Definition 2.2.1: Suppose \(f\) is such that

\[ a_n = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\cos(nx)\mathrm{d}x, \qquad b_n = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\sin(nx)\mathrm{d}x, \]

exist. Then we shall write

\[ f(x)\sim \frac{1}{2}a_0 + \sum_{n=1}^\infty \left[a_n\cos(nx) + b_n\sin(nx)\right], \]

and call the series on the right-hand side the Fourier series for \(f\), whether or not it converges to \(f\). The constants \(a_n\) and \(b_n\) are called Fourier coefficients of \(f\).


Example 2.2.1 Find the Fourier series of the function \(f\) which is periodic of period \(2\pi\) and such that

\[ f(x) = |x|, \quad x\in(-\pi, \pi]. \]

To find \(a_n, b_n\) first notice that \(f\) is even, so that \(f(x)\sin(nx)\) is odd and

\[ b_n = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\sin(nx)\mathrm{d}x = 0, \]

for every \(n\). Also, \(f(x)\cos(nx)\) is even, so

\[\begin{split} a_n &=\frac{1}{\pi}\int_{-\pi}^\pi f(x)\cos(nx)\mathrm{d}x,\\ &=\frac{2}{\pi}\int_{0}^\pi f(x)\cos(nx)\mathrm{d}x,\\ &=\frac{2}{\pi}\int_{0}^\pi x\cos(nx)\mathrm{d}x,\\ &= \frac{2}{\pi}\left(\left[\frac{x\sin(nx)}{n}\right]_0^\pi - \int_0^\pi \frac{\sin(nx)}{n}\mathrm{d}x\right),\\ &=-\frac{2}{\pi}\left(\left[\frac{-\cos(nx)}{n^2}\right]_0^\pi\right),\\ &=\frac{2}{\pi}\left(\frac{\cos(n\pi)-\cos(0)}{n^2}\right),\\ &=\frac{2}{\pi}\frac{((-1)^n-1)}{n^2}. \end{split}\]

Note that this is not valid for \(n=0\). In fact, \(a_0=\pi\).

If \(n\) is even, \(n=2m\) say, we have

\[ a_{2m} = \frac{2((-1)^{2m}-1)}{\pi(2m)^2} = 0. \]

If \(n\) is odd, \(n=2m+1\) say, we obtain

\[ a_{2m+1} = \frac{2((-1)^{2m+1}-1)}{\pi(2m+1)^2} = \frac{-4}{\pi(2m+1)^2}. \]

The Fourier series expansion of the given function is therefore

\[ f(t) \sim \frac{\pi}{2} + \sum_{m=0}^\infty{\frac{-4}{\pi(2m+1)^2}\cos{\left((2m+1)x\right)}}. \]

Below we use Python to plot the FS approximations (with 1, 2 and 3 terms) of \(|x|\) (shown in blue) on the range \((-\pi, \pi)\). We see that it is in excellent agreement.

# python modules (maths + plotting)
import numpy as np
import matplotlib.pyplot as plt
plt.rcParams['figure.figsize'] = [10, 6] # sets the size of the figure in inches

# domain of plot (-pi, pi)
x = np.linspace(-np.pi,np.pi,100)

# square wave function (Example 2.2.2)
f=np.abs(x)

# Partial sums for the FS of the square wave
y1=np.pi/2-(4/np.pi)*np.cos(x) # one term (first and second harmonics)
y2=y1-(4/np.pi)*np.cos(3*x)/9.0 # 2 terms
y3=y2-(4/np.pi)*np.cos(5*x)/25.0 # 3 terms

# plot
plt.ion()
plt.plot(x,f,'b-',lw=3,label='f(x)')
plt.plot(x,y1,'r--',lw=2,label='two terms')
plt.plot(x,y2,'y--',lw=2,label='four terms')
plt.plot(x,y3,'m--',lw=2,label='six terms')

# add legend, label axis and add title
plt.legend(loc='best')
plt.xlabel('x',style='italic')
plt.ylabel('partial sums',style='italic')
plt.suptitle('Partial sums for Fourier series of $f(x) = |x|$ on $(-\pi, \pi)$',size=16,weight='bold')

Sine and cosine series¶

Let \(f\) be \(2\pi\)-periodic. If \(f\) is odd then

\[ f(x)\sim \sum_{n=1}^\infty b_n\sin(nx), \]

where

\[ b_n = \frac{1}{\pi}\int_{-\pi}^\pi f(s)\sin(ns)\mathrm{d}s = \frac{2}{\pi}\int_{0}^\pi f(s)\sin(ns)\mathrm{d}s, \]

i.e. \(f\) has a Fourier sine series. In this case \(a_n=0\) because \(f(x)\cos(nx)\) is odd.

If \(f\) is even then

\[ f(x)\sim \frac{1}{2}a_0 + \sum_{n=1}^\infty a_n\cos(nx), \]

where

\[ a_n = \frac{2}{\pi}\int_0^\pi f(s)\cos(ns)\mathrm{d}s, \]

i.e. \(f\) has a Fourier cosine series.

Question 2¶

Recall question 2: with these \(a_n, b_n\) when, if ever, is equation (1) true? Consider what happens in the following example.


Example 2.2.2 Find the Fourier series of the function \(f\) which is periodic of period \(2\pi\) and such that

Square Wave Function

\[\begin{split} f(x) = \begin{cases} -1, & -\pi<x<0,\\ 1, & 0<x<\pi,\\ \end{cases} \end{split}\]

The function \(f\) is know as the square wave function.


To find the \(a_n, b_n\) first notice that \(f\) is odd, so \(f(x)\cos(nx)\) is odd so that

\[ a_n = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\cos(nx)\mathrm{d}x = 0, \]

for \(n = 0, 1, 2, \ldots\). Also, \(f(x)\sin(nx)\) is even, so

\[\begin{split} b_n &= \frac{1}{\pi}\int_{-\pi}^\pi{f(x)\sin{(nx)}\mathrm{d}x},\\ &= \frac{2}{\pi}\int_{0}^\pi{f(x)\sin{(nx)}\mathrm{d}x},\\ &= \frac{2}{\pi}\int_{0}^\pi{\sin{(nx)}\mathrm{d}x},\\ &=\frac{2}{\pi}\left[-\frac{\cos{nx}}{n}\right]_0^\pi,\\ &=\frac{2}{n\pi}\left(1-(-1)^n\right). \end{split}\]

The Fourier series expansion of the given function is therefore

(6)¶\[ f(t) \sim \sum_{n=1}^\infty{\frac{2}{n\pi}\left(1-(-1)^n\right)\sin{nx}}. \]

Noting that

\[ b_{2m} = 0, \qquad b_{2m+1} = \frac{4}{(2m+1)\pi}, \]

we can reduce equation (6) to

\[ f(x) \sim \frac{4}{\pi}\sum_{m=0}^\infty \frac{1}{2m+1}\sin\left((2m+1)x\right). \]

Consider question 2 for this case: when is

\[ f(x) = \frac{1}{2}a_0 + \sum_{n=1}^\infty \left[a_n\cos(nx)+b_n\sin(nx)\right]? \]

Recall that

\[ \frac{4}{\pi}\sum_{m=0}^\infty\frac{\sin\left((2m+1)x\right)}{2m+1} \]

means \(\displaystyle \lim_{n\to\infty} s_n(x)\) where

\[ s_n(x) = \frac{4}{\pi}\sum_{m=0}^n\frac{\sin\left((2m+1)x\right)}{2m+1}. \]

The question is therefore, does \(s_n(x)\) converge for each \(x\)? If it does, is the limit \(f(x)\)?


The python script below plots an approximation of the square wave function (shown in blue) consisting of 2, 4 and 6 terms of the Fourier series in (6). We see that the FS appears to be converging to the function as we increase the number of terms although not as quickly as the series in Example 2.2.1. In particular, there appears to be an issue at zero…

Note

The slower convergence shouldn’t be too surprising given that the square wave function is discontinuous at \(x=0\). We shall return to this issue later on in the course.

# python modules (maths + plotting)
import numpy as np
import matplotlib.pyplot as plt
plt.rcParams['figure.figsize'] = [10, 6] # sets the size of the figure in inches

# domain of plot (-pi, pi)
x = np.linspace(-np.pi,np.pi,100)

# square wave function (Example 2.2.2)
f=np.ones_like(x)
f[x<0]=-1

# Partial sums for the FS of the square wave
y1=(4/np.pi)*(np.sin(x)+np.sin(3*x)/3.0) # two terms (first and second harmonics)
y2=y1+(4/np.pi)*(np.sin(5*x)/5.0+np.sin(7*x)/7.0) # 4 terms
y3=y2+(4/np.pi)*(np.sin(9*x)/9.0)+np.sin(11*x)/11.0 # 6 terms

# plot
plt.ion()
plt.plot(x,f,'b-',lw=3,label='f(x)')
plt.plot(x,y1,'c--',lw=2,label='two terms')
plt.plot(x,y2,'r--',lw=2,label='four terms')
plt.plot(x,y3,'g--',lw=2,label='six terms')

# add legend, label axis and add title
plt.legend(loc='best')
plt.xlabel('x',style='italic')
plt.ylabel('partial sums',style='italic')
plt.suptitle('Partial sums for Fourier series of the square wave function',size=16,weight='bold')
